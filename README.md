# Введение

Компьютерное зрение играет ключевую роль в обработке визуальной информации, позволяя автоматизировать процессы анализа изображений и распознавания объектов. Данная выпускная квалификационная работа посвящена разработке модели компьютерного зрения для обработки фотоснимков страниц периодических изданий с целью решения двух важных задач: исключение дубликатов страниц и группировку выпусков.

## Актуальность исследования

Обработка большого объема цифровых материалов становится актуальной задачей в эпоху цифровизации архивов и библиотек. Автоматическое выявление дублирующихся страниц позволяет оптимизировать хранение и управление информацией, предотвращая избыточность данных. Группировка выпусков помогает систематизировать архивы, облегчая последующий поиск и навигацию.

## Цель и задачи исследования

Целью работы является создание эффективной модели компьютерного зрения, способной качественно решать поставленные задачи. Для достижения цели необходимо решить следующие задачи:

1. Исключение дубликатов: разработка метода, позволяющего выявлять и удалять повторяющиеся страницы, сохраняя только качественные экземпляры.

2. Группировка выпусков: создание алгоритма, обеспечивающего точное разделение снимков страниц по соответствующим выпускам.

## Предположительные подходы и методы

Исследование предполагает использование классических методов компьютерного зрения, таких как SURF, для начального этапа выявления дубликатов. Для повышения точности планируется применение глубоких архитектур, включая ResNet и трансформеры, а также технологии оптического распознавания символов (OCR).

Предварительная обработка данных включает обнаружение контуров страниц, выравнивание ориентации и коррекцию искажений, вызванных физическими свойствами материала (коробление бумаги). Эти шаги необходимы для улучшения качества исходных снимков перед применением моделей машинного обучения.

## Ожидаемые результаты

Результатом работы должна стать эффективная модель, способная обрабатывать большое количество фотоснимков страниц периодических изданий, обеспечивая высокую точность и эффективность в решении поставленных задач. Модель будет полезна для автоматизации процессов управления цифровыми архивами и библиотеками, улучшая доступность и удобство использования хранящихся данных.

Таким образом, данная работа направлена на разработку инновационного подхода к решению проблем, возникающих при работе с большими объемами цифровой информации, что имеет важное значение для современного общества и науки.
